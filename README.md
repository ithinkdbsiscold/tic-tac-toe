# Tic Tac Toe

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Turn-based gameplay
- Winner detection
- Tie detection
- Reset game functionality
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open index.html in your browser

## Future Improvements
- Score tracking
- Multiplayer mode
- Dark mode
